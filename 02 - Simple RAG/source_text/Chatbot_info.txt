Syllabus Info:

Unit I: Introduction and Operating Systems structures
Evolution of operating systems: Batch, timesharing, multiprogramming, multitasking and distributed and real time. Operating system components, O.S. Services,
System Calls, System Programs, System Structure, Virtual Machines, Special
purpose operating systems, Open-source operating systems, Boot Procedure,
Overview of the GNU/Linux system administration, Comparison of various
operating systems

Unit II: Process Management and Scheduling Algorithms
Processes: Definition, Process Relationship, Different states of a Process, Process
State transitions, Process Control Block (PCB), Context switching. Process
Scheduling: Foundation and Scheduling objectives, Types of Schedulers,
Scheduling criteria: CPU utilization, Throughput, Turnaround Time, Waiting Time,
and Response Time. Scheduling algorithms: Pre-emptive and non-pre-emptive,
FCFS, SJF, RR

Unit III: Process Synchronization, Threads and Deadlocks
Inter-process Communication: Concurrent processes, Critical Section, Race
Conditions, Mutual Exclusion, Hardware Solution, Semaphores, The Producer /
Consumer Problem, Monitors, Message Passing, Classical IPC Problems: Reader‟s
& Writer Problem, Dinning Philosopher Problem.
Thread: Definition, Various states, Benefits of threads, Types of threads, Concept
of multithreads, Introduction to POSIX library.

Unit IV: Memory Management
Basic Memory Management, Static and Dynamic memory allocation, contiguous
and non-contiguous memory allocation, Paging, Segmentation, Page replacement
algorithms.

Unit V: I/O and Device Management
File Management: Concept of File, Access methods, File types, File operation,
Directory structure, File System structure, Allocation methods (contiguous, linked,
indexed), Free-space management (bit vector, linked list, grouping),
Disk Management: Disk structure, Disk scheduling - FCFS, SSTF, SCAN, CSCAN, Disk reliability, Disk formatting, Boot-block, Bad blocks.

Exam Structure:{
🎓 **Teaching Mode & Duration**

* Teaching Mode: Offline
* End Semester Exam (ESE) Duration: 2.5 Hours

🎯 **Credits & Marks Distribution**

* Total Credits: 4
* Marks Breakdown:

  * TAE (Teacher Assessment Exam): 20 marks
  * CAE (Class Assessment Exam): 20 marks
  * ESE (End Semester Exam): 60 marks
  * Practical (Internal): 25 marks
* **Total Marks:** 125 marks

✅ **Abbreviations**

* TAE → Teacher Assessment Examination
* CAE → Class Assessment Examination
* ESE → End Semester Examination

📝 **CAE Exam Structure**

* Total Major Questions: 4
* Student must attempt **any 2 compulsory major questions**
* Each Major Question has **2 sub-questions (both mandatory)**
* Each Sub-question is worth 5 marks
* **Important Rule:**

  * Only get 10 marks for a major question if **both sub-questions are solved**
  * Solving only one sub-question → 0 marks for that major question
* **Total CAE Marks:** 20 per CAE
* **Frequency:** 2 CAE exams per semester
* **Final CAE:** Average of the two CAEs → Final CAE = 20 marks

🧑‍🏫 **TAE Structure**

* **TAE1:** MCQ + Short Answer type
* **TAE2:** Poster on a topic given by faculty (Topic should be relevant to syllabus)
* Both TAE exams scored out of 20 → average taken → Final TAE = 20 marks

📌 **ESE Exam Structure**

* Total Major Questions: 5
* Each Major Question has multiple sub-questions (all mandatory)
* **Exception:** When “OR” is present between two sub-questions
* Marks per sub-question: Mostly 4, 3, or 6 marks each
* **Important:** Neat diagrams must be drawn wherever necessary
* **Total Marks:** 60 marks

⚠️ **Note:**

* ESE Exam Structure is based on **Winter 2024 paper**
* May vary slightly depending on Exam Organizing Committee
* Always confirm with teacher if Winter 2024 format is used for current exam
